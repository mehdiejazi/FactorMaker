# پست لینکدین - FactorMaker

چند وقت پیش یک پروژه شخصی برای مدیریت فروشگاه و صدور فاکتور را شروع کرده بودم و حالا نسخه اوپن‌سورس آن را منتشر کرده‌ام.

`FactorMaker` یک نمونه full-stack برای این سناریوهاست:

- مدیریت فروشگاه
- مدیریت دسته‌بندی و کالا
- مدیریت مشتری
- صدور و مدیریت فاکتور
- مدیریت کاربران و نقش‌ها
- داشبورد فروش مبتنی بر داده واقعی

Stack این پروژه:

- ASP.NET Core Web API
- Angular
- SQL Server
- JWT Authentication
- Repository / UnitOfWork Pattern

برای اینکه ریپو فقط یک نمونه کدنویسی نباشد، مسیر راه‌اندازی و seed داده واقعی‌نما هم داخل پروژه گذاشته‌ام تا بتوانید سریع اجرا کنید، flowها را ببینید و توسعه بدهید.

اگر روی این حوزه کار می‌کنید یا درباره ساختار پروژه، طراحی API و تجربه پنل ادمین نظر دارید، خوشحال می‌شوم بازخوردتان را بدانم.
